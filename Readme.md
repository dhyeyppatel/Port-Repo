# Portfolio ChatBot

An interactive, terminal-inspired Chatbot Portfolio built to showcase projects, skills, and freelance services dynamically. Instead of a traditional scrolling website, visitors can interact with a simulated chatbot and terminal to explore details.

## 🌟 Features
- **Interactive Chat Interface**: Type commands or click quick-action chips to navigate.
- **Glassmorphism Design**: A sleek, modern UI with a warm amber/orange dark theme and smooth micro-animations.
- **Dynamic Terminal Panel**: The right-side panel acts as a code preview, but can also swap to host mini-games or web tools.
- **Freelance Onboarding Flow**: Direct lead generation that dynamically pre-fills a Gmail compose window based on the client's needs.
- **GitHub Gist Integration**: Automatically fetches and embeds public GitHub gists.
- **Resume Viewer**: A built-in modal to view and download the resume.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Outfit, JetBrains Mono
- **Hosting**: Designed for easy static deployment (GitHub Pages, Cloudflare Pages, Vercel, etc.)

## ⌨️ Chat Commands
You can interact with the bot by typing the following commands in the input box:

### Standard Commands
- `help` - Lists all available commands.
- `about` - Scrolls to the About section and shares background information.
- `projects` - Highlights the projects grid and shows the search filter.
- `resume` - Opens the Resume viewer modal.
- `blogs` - Shows the integrated GitHub gists.
- `contact` - Scrolls to the Contact details.

### Interactive Features
- `freelance` (or `hire me`) - Initiates the Freelance Hiring flow, asking what kind of project you need and generating a pre-filled Gmail draft.
- `play` - Replaces the terminal with a playable mini-game (Tic-Tac-Toe).
- `tools` - Replaces the terminal with a fully functional Mini Toolbox (e.g., Text Case Converter).

### 🤫 Easter Eggs
- `sudo run` (or `sudo`, `matrix`) - Grants "Access", transforming the entire portfolio UI into a green Matrix-style hacker terminal.
- `coffee` - The bot serves you a hot cup of ASCII coffee to keep you coding.
